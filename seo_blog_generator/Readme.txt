# AI-Powered SEO Blog Generator

## Introduction
Welcome to the AI-Powered SEO Blog Generator! This project automates the process of researching, planning, writing, optimizing, and proofreading blog content, ensuring high-quality, SEO-friendly articles effortlessly.

## Features
- **Trending Topic Research**: Automatically fetches the latest HR trends.
- **Content Planning**: Generates a structured blog outline.
- **Content Generation**: Writes a detailed blog post based on the outline.
- **SEO Optimization**: Enhances content with relevant keywords and readability improvements.
- **Proofreading**: Checks and corrects grammar and sentence structure.
- **File Saving**: Saves the blog as both Markdown and HTML formats.

## How It Works
1. **Research Agent**: Fetches trending HR topics.
2. **Content Planning Agent**: Generates an outline for the selected topic.
3. **Content Generation Agent**: Expands the outline into a full blog post.
4. **SEO Optimization Agent**: Improves the blog for better search engine visibility.
5. **Review Agent**: Proofreads and corrects any mistakes.
6. **Saving Process**: Saves the final blog in Markdown and HTML formats.

## Installation & Usage
### Prerequisites
- Python 3.7+
- Required Python Libraries: `requests`, `textblob`, 'g4f'

### Setup
1. Clone this repository:
   ```sh
   git clone https://github.com/ReshmaMasutha/seo_blog_generator.git
   cd seo_blog_generator.git
   ```
2. Install dependencies:
   ```sh
   pip install -r requirements.txt
   ```
3. Run the script:
   ```sh
   python main.py
   ```
